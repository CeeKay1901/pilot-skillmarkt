# Story-Muster für Daten-Slides

## Der rote Faden
Jedes Daten-Deck folgt drei Bewegungen:
1. **Situation** — wo stehen wir? (1 Slide, der Rahmen)
2. **Erkenntnis** — was zeigen die Zahlen? (die Kern-Slides, je eine Aussage)
3. **Empfehlung** — was tun wir daraus? (1 Slide, konkret)

## Eine Slide, eine Botschaft
Der Titel ist die Aussage, nicht das Thema:
- ✗ "Paid Social Ergebnisse"
- ✓ "Paid Social treibt den Erfolg — ROAS 4,2"

## Chart folgt Aussage
| Aussage-Typ            | passender Chart      |
|------------------------|----------------------|
| Vergleich (A vs. B)    | Balken               |
| Entwicklung über Zeit  | Linie                |
| Anteil am Ganzen       | gestapelt / Donut    |
| Zusammenhang           | Streu / Bubble       |
| eine große Zahl        | Kennzahl groß setzen |

## Sprechernotiz
Ein Satz pro Slide — das, was man live dazu sagt und was nicht auf der Slide steht.
