# Slides aus Daten

## Ablauf
1. Daten einlesen, Kernaussagen finden (2–5).
2. Pro Aussage eine Slide: Botschaft + Chart-Vorschlag.
3. Roter Faden: Situation → Erkenntnis → Empfehlung.
4. Kurze Sprechernotiz je Slide.

## Prinzip
Eine Slide = eine Botschaft.
