# Slides aus Daten

Macht aus einer Tabelle eine Geschichte: nicht "hier sind die Zahlen", sondern "das sagen sie uns".

## Wann ich starte
"Bau mir Slides aus dieser Tabelle", Reporting-Aufbereitung, vor einem Kunden- oder Team-Termin.

## Vorgehen
1. **Daten einlesen** und die 2–5 Kernaussagen finden — was ist wirklich die Nachricht?
2. **Pro Aussage eine Slide**: Botschaft als Titel, dazu der passende Chart-Typ (siehe `references/story-muster.md`).
3. **Roten Faden legen**: Situation → Erkenntnis → Empfehlung.
4. **Sprechernotiz** je Slide: der Satz, den man live dazu sagt.

## Ausgabe
Slide-Struktur mit Titeln, Chart-Vorschlägen und Sprechernotizen (siehe `references/beispiel.md`) — als Gerüst zum Ausgestalten.

## Regeln
- Eine Slide = eine Botschaft. Lieber eine Aussage weglassen als überladen.
- Chart folgt der Aussage, nicht umgekehrt — kein Diagramm ohne Kernsatz.
- Nichts behaupten, was die Daten nicht hergeben.
