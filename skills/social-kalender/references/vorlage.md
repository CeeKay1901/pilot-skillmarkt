# Redaktionsplan — Vorlage & Kanal-Spickzettel

## CSV-Kopf
```
Datum;Kanal;Format;Thema;Hook;Hashtags
```

## Format je Kanal (Orientierung)
| Kanal      | starke Formate           | Frequenz (Orientierung) |
|------------|--------------------------|-------------------------|
| LinkedIn   | Text-Post, Carousel, Doc | 2–4×/Woche              |
| Instagram  | Reel, Carousel, Story    | 3–5×/Woche              |
| TikTok     | kurzes Reel / Video      | 3–7×/Woche              |
| Newsletter | Teaser + Link            | wöchentlich–14-tägig    |

## Gute Hooks (konkret statt "spannender Post")
- Zahl / Ergebnis: "480.000 € Budget — hier ging es hin"
- Fehler / Warnung: "3 Fehler, die jeden Mediaplan killen"
- Behind-the-Scenes: "Wie ein Pitch bei uns wirklich abläuft"
- Frage / These: "Reichweite ist überbewertet. Ein Konter."

## Regeln
- Rhythmus halten, Anlässe (Feiertage, Launches) einplanen, nicht überladen.
- Ein Post = ein Ziel. Hashtags kanaltypisch dosieren.
