# dataviz

Lies BEVOR du die erste Chart-Zeile schreibst `references/palette.md`.

## Ablauf
1. Daten verstehen (Spalten, Typen, Größenordnung).
2. Passende Darstellungsform wählen (Heuristik unten).
3. Farben nach Palette + Kontrast-Validator setzen.
4. Als eigenständiges HTML (inline CSS/JS) oder Bild ausgeben.

## Form-Heuristik
- Vergleich → Balken · Zeitverlauf → Linie · Anteil → gestapelt/Donut
- Verteilung → Histogramm · Zusammenhang → Scatter
