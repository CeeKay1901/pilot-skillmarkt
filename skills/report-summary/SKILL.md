# Report Summarizer

Optimiert für pilot-Reporting-Formate: Media-Performance, Marktanalysen, Wettbewerbsanalysen. Destilliert den Kern, hebt die wichtigsten Insights hervor und formuliert nächste Schritte in klarem Deutsch — aus 30 Seiten wird eine Seite, die man wirklich liest.

## Auslöser — wann ich genutzt werde
Nutze diesen Skill für: Wöchentliche Performance-Reports · Marktanalysen · Wettbewerbsanalysen · Management-Summaries.
Stichworte: analytics, content, productivity, reports, summarization.

## Vorgehen
1. Eingabe/Kontext erfassen (Datei, Text oder Auftrag).
2. Daten lesen, bereinigen und die relevanten Muster herausarbeiten.
3. Ergebnis strukturiert erzeugen (siehe Ausgabe).
4. Kurz zusammenfassen, was getan wurde.

## Ausgabe
Bereinigte Datei bzw. kompakte Auswertung mit Kernzahlen.

## Regeln
- Klar und für Nicht-Techniker:innen verständlich bleiben.
- Nichts erfinden — nur, was aus der Eingabe hervorgeht.
- Bei Unklarheit kurz nachfragen statt raten.
