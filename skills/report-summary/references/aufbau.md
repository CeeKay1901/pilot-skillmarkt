# Executive Summary — Aufbau & Verdichtung

## Struktur
```
## Executive Summary — <Report>
<2–4 Sätze: das Wichtigste zuerst — was ist das Ergebnis?>

▸ Key Insights (3–5)
- <Aussage mit Zahl> — <warum es zählt>

▸ Empfehlungen (priorisiert)
- <konkret, umsetzbar, mit Richtung>
```

## Verdichtungs-Fragen (je Kernaussage)
1. **Was** ist passiert? (die Zahl / der Fakt)
2. **Warum** ist es passiert? (Ursache, wenn belegbar)
3. **Was heißt das** für uns? (Konsequenz / Handlung)

## Do
- Das Wichtigste in den ersten zwei Sätzen (Pyramiden-Prinzip).
- Jede Zahl aus der Quelle belegt.
- Entscheidungsorientiert: Leser:in weiß danach, was zu tun ist.

## Don't
- Keine erfundenen Zahlen, keine Schönung durch Runden.
- Keine Nacherzählung des ganzen Reports — Summary, nicht Vollprotokoll.
