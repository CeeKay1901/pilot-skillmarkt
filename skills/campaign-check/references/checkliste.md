# Campaign-Check — Prüfpunkte

## Tracking
- [ ] Conversion-Tracking aktiv (GA4 / Pixel) und getestet
- [ ] Events feuern korrekt, keine Doppelungen
- [ ] Consent-Mode / Einwilligung berücksichtigt

## UTM
- [ ] Konvention eingehalten (source/medium/campaign konsistent, lowercase)
- [ ] Alle Anzeigen/Links getaggt
- [ ] Keine Tippfehler / Sonderzeichen

## Zielgruppe
- [ ] Definitionen vollständig, Overlap gering (< ~20%)
- [ ] Ausschlüsse gesetzt (Bestandskunden etc.)

## Budget
- [ ] Summen stimmen, Caps gesetzt (Kampagne/Ad-Set)
- [ ] Flighting ohne ungewollte Lücken

## KPI
- [ ] Ziel-KPIs definiert und messbar
- [ ] Landingpages erreichbar (kein 404), passend zur Anzeige
