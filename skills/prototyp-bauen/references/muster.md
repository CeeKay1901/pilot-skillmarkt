# Erprobte Muster

## Rechner
Eingabefelder + Live-Ergebnis. Ergebnis bei jeder Eingabe neu berechnen.

## Formular
Felder + Validierung + "Absenden" → Zusammenfassung anzeigen.

## Tabelle / Vergleich
Zeilen dynamisch hinzufügen, Spalten summieren, sortierbar.

## Design
pilot-Farben (Schwarz #262626, Gelb #ffe05e, Beige #f1f1ec),
großzügiger Weißraum, klare Hierarchie, mobil bedienbar.
