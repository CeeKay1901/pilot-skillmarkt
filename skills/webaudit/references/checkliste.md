# webaudit — Prüf-Checkliste

## SEO
- [ ] <title> vorhanden & aussagekräftig (< 60 Zeichen)
- [ ] Meta-Description vorhanden (< 155 Zeichen)
- [ ] Genau eine <h1>, sinnvolle H2/H3-Hierarchie
- [ ] Open-Graph-Bild + Titel/Description fürs Teilen
- [ ] Sprechende, funktionierende Links (keine 404)

## Barrierefreiheit (WCAG)
- [ ] Alle Bilder mit sinnvollem alt-Text
- [ ] Textkontrast ≥ 4.5:1
- [ ] Bedienbar per Tastatur, sichtbarer Fokus
- [ ] Formularfelder mit Labels

## Performance
- [ ] Bilder komprimiert & richtig dimensioniert
- [ ] Keine blockierenden Riesen-Skripte
- [ ] Mobile Ladezeit vertretbar (< ~2.5s LCP)

## UX / Design
- [ ] Klarer CTA "above the fold"
- [ ] Konsistente Typo & Abstände
- [ ] Mobil bedienbar, nichts läuft über
