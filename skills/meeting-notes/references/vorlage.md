# Protokoll-Vorlage

Immer diese Struktur — leere Abschnitte weglassen, nichts erfinden.

```
## Protokoll — <Anlass> (<Datum>)

Teilnehmer: <Kürzel/Namen>
Ort/Format: <vor Ort / Call>

### Agenda
1. …

### Beschlüsse
- <was wurde entschieden — knapp, eindeutig>

### Action Items
- [ ] <wer>: <was> — bis <wann>
- [ ] <wer>: <was> — bis <wann>

### Offene Punkte
- <ungeklärt / vertagt> [?]

### Nächste Schritte
- <nächster Termin / Meilenstein>
```

## Regeln beim Ausfüllen
- Jedes Action Item hat eine:n Verantwortliche:n. Fehlt → `[?]`, nicht weglassen.
- Fristen konkret (Datum), nicht "bald".
- Beschluss ≠ Diskussion: nur festhalten, was wirklich entschieden wurde.
