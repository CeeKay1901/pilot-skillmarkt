# Moodboard-Generator

## Ablauf
1. Briefing lesen: Marke, Zielgruppe, Botschaft.
2. Stimmung in 3–5 Adjektiven verankern.
3. Farbwelt, Bildsprache, Typo-Gefühl ableiten.
4. Konkrete Referenz-Stichworte zum Suchen liefern.

## Wichtig
Kein fertiges Design — ein Gerüst, das ausrichtet.
