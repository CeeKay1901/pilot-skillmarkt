Umfrage-Auswerter

CSV einfügen → Antwortverteilung als Balken. Erste Spalte = Frage, weitere Spalten = Antworten.

Format je Zeile: Frage;Antwort — eine Zeile pro abgegebener Antwort. Trennzeichen ; oder , (pro Datei einheitlich). Kommas in Antworten in „Anführungszeichen" setzen. Eine Kopfzeile wird automatisch erkannt.

Deine Daten bleiben lokal in deinem Browser — die Auswertung rechnet direkt hier, es wird nichts hochgeladen oder gesendet.
